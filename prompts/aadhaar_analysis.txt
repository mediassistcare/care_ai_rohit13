1. Extract all visible text and personal details from the Aadhaar card
2. Identify and extract the following fields if visible:

   - Date of Birth (in DD/MM/YYYY or DD-MM-YYYY format)
   - Gender (Male/Female)
   - Address (complete address)
   - Aadhaar Number (12-digit number)
   
3. Handle both front and back sides of Aadhaar card
4. If the image contains both sides, extract information from both
5. Clean and format the extracted data appropriately

- Extract ONLY what is clearly visible and readable

- For addresses, extract the complete address as shown
